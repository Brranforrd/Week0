Week 0 Review

1. I learned about computational thinking

Computational thinking is a form of abstract and algorithmic thinking that involves decomposition, pattern recognition, and debugging. With computational thinking, I am able to become a better problem solver and approach coding challenges more efficiently.

2. I learned the basics of maneuvering GitHub

GitHub is a type of version control system that stores files and tracks edits over time. GitHub is highly popular and useful to the majority of coding developers.

3. I learned what a repository is

A repository is the storage space where project files are held. Repositories are essential for keeping and storing different lines of code.

4. Using VSCode

Using VSCode has been extremely beneficial and user-friendly. VSCode is my favorite coding editor to use because it is quick and simple.

5. Committing, pushing, and pulling origin

These tools are especially helpful when it comes to saving your code. Committing means taking a snapshot of your changes, and pushing and pulling allow you to sync your local repository with the remote one. Learning how to navigate between using these tools has improved my workflow.
